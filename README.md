### config 항목 정리
"safe_level"
- 0: default, fence 미적용 상태 
- 1: 안전 (초록색)
- 2: 1차 감지 (노란색)
- 3: 2차 감지 (빨간색)

"recording" : 녹화 상태값 (녹화중/녹화정지)

"area" 
- "area_active": 영역 활성화 상태 표기(default=false)
- "area_edit": 영역 수정 상태 표기(default=false)
- "area_number": 영역 이름
- "primary_detection":  해당 영역에서 1차 감지된 횟수
- "secondary_detection": 해당 영역에서 2차 감지된 횟수
